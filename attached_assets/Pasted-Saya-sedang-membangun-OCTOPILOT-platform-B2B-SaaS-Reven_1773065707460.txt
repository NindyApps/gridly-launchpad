Saya sedang membangun OCTOPILOT — platform B2B SaaS Revenue Signal Intelligence di Replit.

═══ PRODUK ═══

═══ REPLIT-SPECIFIC SETUP ═══

- Replit Agent dipakai untuk generate kode per fase
- IDE Replit dipakai untuk edit manual, debug, dan review kode
- Run command: npm run dev (development), npm run build && npm start (production)
- Port: Replit otomatis expose port 3000
- File .env.local hanya untuk local dev — production pakai Replit Secrets
- Replit DB tidak dipakai — semua data di Supabase
